# CLI-Countries Trivia-Game

## Introduction
CLI Countries Trivia is a program that can be run on your VSCode as a **Command Line Interface**. In this trivia, you can challenge your knowledge of countries based on continents and answer a quiz of the countries in a content.

## Instructions

### Setup instructions

- Open the project in VSCode.
- Run `npm install` to install all necessary dependencies.
- After installing the necessary dependencies, type `npm link`.
- Use the word `countries-trivia` to start the game on your computer.

## Planned functionality
- Ability to save your statistics from a game such as the score, date and type of game mode.
- Capability to engage in a `challenge mode`, which issues a countdown time, and then expects a completion of the quiz within the timeframe.
- A **game mode**, involving a quiz of answering capital cities by selecting one of 4 choices from a multiple choice question.
